# 🏆 TierZenith official website

Welcome to **TierZenith**, a highly optimized, fully responsive competitive Minecraft PvP Leaderboard and Tier List platform.tracks top-tier players, gamemodes, and competitive rankings across different regions.

## 🚀 Live Demo
You can view the live website here:

## ✨ Features
* 🎮 **Multiple Gamemodes:** Seamlessly switches context between Vanilla, UHC, Pot, NethOP, SMP, and Sword.
* 🌍 **Region Tracking:** Dedicated tags for players across NA, EU, and AS.
* 🥇 **Tier System:** Highly accurate structural tiers ranking from HT1 (High Tier 1) down to LT3 (Low Tier 3).
* ⚡ **Ultra Responsive Design:** Dark-themed UI crafted meticulously for mobile, tablet, and desktop viewing.

---

## 🛠️ Tech Stack Used
* **HTML5:** Structuring the core elements and player ranking data blocks.
* **CSS3:** Custom modern property variables, sleek glassmorphic components, and dynamic hover layouts.
